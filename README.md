Handwritten Digit Recognition using Machine Learning Algorithms
This project utilizes machine learning algorithms to recognize handwritten digits. The algorithms implemented include:

- Support Vector Machine (SVM)
- Random Forest Classifier
- Decision Tree Classifier
- K-Nearest Neighbors (KNN)

The project aims to compare the performance of these algorithms on the MNIST dataset.

Features
- Implementation of multiple machine learning algorithms for handwritten digit recognition
- Comparison of algorithm performance on the MNIST dataset
- Use of Python and scikit-learn library for implementation

Requirements
- Python 3.x
- scikit-learn library
- MNIST dataset

Usage
1. Clone the repository
2. Install required libraries using pip
3. Run the code using Python
